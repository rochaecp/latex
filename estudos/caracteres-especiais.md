# Caracteres Especiais

| Caractere | Descrição         | Código                | Explicação                |
| :---      | :---              | :---                  | :---                      |
| ç         |                   | ```\c c```            |                           |
| ã         |                   | ```\~a```             |                           |
| ó         |                   | ```\'o```             |                           |
| ô         |                   | ```\^o```             |                           |
| 2°        |                   | ```2$^\circ$```       |                           |
| "         | Abre aspas        | ``` `` ```            | usar sinal de crase       |
| "         | Fecha aspas       | ``` '' ```            | usar duas aspas simples   |
| \n        | Quebra de linha   | ``` \\ ```            |                           |
| _         | Underline         | ``` \_ ```            |                           |
| $         |                   | ``` \$ ```            |                           |
| #         |                   | ``` \# ```            |                           | 
| %         |                   | ``` \% ```            |                           | 
| ^         |                   | ``` \^ ```            |                           | 
| &         |                   | ``` \& ```            |                           | 
| {         |                   | ``` \{ ```            |                           |
| }         |                   | ``` \} ```            |                           | 
| ~         |                   | ``` \~ ```            |                           | 
| \         |                   | ``` $\backslash$ ```  |                           |
| <         |                   | ``` \textless ```     |                           |
| >         |                   | ``` \textgreater ```  |                           |
